## Barocco key settings

#### 基本操作

| 操作 | 対象キー |
|--|--|
| 矢印キー設定 | `Win` + `Right Alt` + `Right Space` |
| レイヤー適用 | `Fn` + `,` |
| 設定開始 | `Fn` + `Right Ctrl` |
| 変更 | (設定モードで) `対象キー` -> `割り当て先` -> `Pn` |
| リセット | `Fn` + `R`(5 sec) |

#### 変更

| 対象キー(変換前) | 割り当て先(変換後) |
|--|--|
| `Right Space` | `Right Shift`(SKK変換用) |
| `Capslock` | `Left Ctrl` |
