## pip upgrading

#### versoion confirmnation

```
$ pip list --o
```

#### Upgrade

Add option `-U`

```
$ pip install <package name> -U
```

#### Upgrade all packages using pip-review

Install

```
$ pip install pip-review
```

Usage

```
$ pip-review --auto
```

Usage (with interactive)

```
$ pip review --interactive
```
