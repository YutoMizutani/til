## Install Homebrew

Required `gem (ruby, rbenv)`

```
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew upgrade --cleanup
$ brew install caskroom/cask/brew-cask
```
