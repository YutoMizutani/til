## Install Homebrew

Required `gem (ruby, rbenv)`

```
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew upgrade --cleanup
$ brew install bash
$ export HOMEBREW_CASK_OPTS="--appdir=/Applications" >> .bashrc
$ source .bashrc
```
