## Install Swift for Raspbian

#### Raspberry Pi 3

```
$ curl -s https://packagecloud.io/install/repositories/swift-arm/release/script.deb.sh | sudo bash
$ sudo apt install swift5
```

#### Raspberry Pi zero

```
$ curl -s https://packagecloud.io/install/repositories/swift-arm/release/script.deb.sh | sudo bash
$ sudo apt install swift4rpi01
```
