## Resizing

#### Resize to (width: 320, height: 240)

```swift
let result = image.resizedTo(width: 320, height: 240)
```

#### Nearest neighbor

```swift
let result = image.resizedTo(width: 320, height: 240, interpolatedBy: .nearestNeighbor)
```
