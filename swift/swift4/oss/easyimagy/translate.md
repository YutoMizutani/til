```swift
import EasyImagy
```

#### UIImage to Image

```swift
let image: Image = Image<RGBA<UInt8>>(uiImage: <#UIImage#>)
```

#### Image to UIImage

```swift
let uiImage: UIImage = <#Image#>.uiImage
```
