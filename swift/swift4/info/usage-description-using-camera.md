## Usage Description using camera

iOS11から，使用時に`Info.plist`に対して明示的な許可が必要になった。カメラ機能の利用に必要な変更は2つ。

| Key | 説明 |
|---|---|
| Privacy - Camera Usage Description | カメラ使用許可 |
| Privacy - Photo Library Usage Description | 画像保存許可 |
