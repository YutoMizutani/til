## Constants and Variables

#### Constants

Swiftにおける定数の宣言キーワードは`let`を用いる。

```swift
let num = 0
num = 1 // Error
```

#### Variables

Swiftにおける変数の宣言キーワードは`var`を用いる。

```swift
var num = 0
num = 1 // OK
```

#### Multiple constants or multiple variables

同時に複数の定数や変数を宣言することもできる。

```swift
let a = 0, b = 1, c = 2
var x = 0, y = 1, z = 2
var red, green, blue: Double
```
