## Create pull request within local repository

> #### 1.ローカルリポジトリにcloneして、ブランチ作って、編集してコミットして、githubにブランチをpushする
>
> ```
> $ git clone リボジトリ
> $ git checkout -b ブランチ名
> ~ ファイルの編集 ~
> $ git commit -a -m 'コメント'
> $ git push origin ブランチ名
> ```
>
